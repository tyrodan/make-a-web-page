Lesson 2: Webpages, Documents, and Structure

Comparison to a house

-HTML is the structure like the walls and rooms.
-CSS is the style of a home like the carpeting and the color of the walls.
-JAVA are the interactive components like a garage door opener or a remote control.

Tree Structure

A web page format is a "tree-like structure". Elements are drawn like a family tree. Elements are encompassed within elements like a box inside a box inside a box.

Scratchpad vs. CodePen vs. Sublime (Programming text editor comparison)

Scratchpad- is best for beginners and provides intermediate feedback. It supports HTML & CSS, but files cannot be shared.
Codepen- requirese intermediate skill and provides feedback in 2-3 seconds. It supports HTML, CSS, and JAVA and is easy to share.
Submlime- advance skill and provides feedback on refresh. It is diificult to share but supports all languages and has the highest professionalism.

Lesson 3: Using CSS Style

CSS

Cascading Style Sheets (CSS) are used for describing the look & formating of a document. It can cascade rules to applied elements and child elements. This allows programmers to avoid repetition in applying style.
